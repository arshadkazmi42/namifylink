Get domain name from url

## Install

```
npm i namifylink
```

## Usage

```javascript

const namifyLink = require('namifylink');

const fileName = namifyLink('https://github.com/arshadkazmi42');
console.log(fileName);

// Output
// github-com


const fileName = namifyLink('https://mail.google.com/arshadkazmi42');
console.log(fileName);

// Output
// mail-google-com

```
